On this page you can create a custom trade AI logic.
If you want to use it you will have to toggle it here and
activate automated trade in the ingame diplomacy window.
Whenever the game calculates your trades it will use these values.

Daily surplus

  • if you are below the desired stockpile this will be the desired daily surplus
  • if you are above the stockpile this value is ignored

Sell above

  • if you are above this then the AI will consider selling resources
  • AI will sell up to a deficit of 'maxDailyDeficit'

Stockpile

  • the desired amount to have in the stockpile
  • if you are above this value (+10%) all trades will be canceled

Money has no desired stockpile, instead it will always try to achieve 100-150% of the surplus.
There is no limit on how much supplies the AI will try to sell, so be careful that you don't sell all your IC.
