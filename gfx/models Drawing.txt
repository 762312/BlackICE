PAINT.NET Instructions

    With painting/model:
        Use "Unit template"
        Cut model background
        Add model in new layer over template
        Scale model to template using "Bicubic" filtering/quality
        Apply Object Shadow to model with Shadow Radius 10 and other values default
        Merge Layers

    With B&W photo:
        Colorize photo (use one of many online Deep Learning services) (https://hotpot.ai/colorize-picture)
        Use "Unit template"
        Add photo in new layer over template and position it
        Use eraser around photo with high brush width (40), 0% Hardness and with Antialiasing
        Merge Layers
